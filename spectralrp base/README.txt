The SpectralRP base was developed by TheRandomnessGuy (https://github.com/TheRandomnessGuy).
The snowScript function (which is disabled by default) requires Atmos, which can be found here: https://steamcommunity.com/sharedfiles/filedetails/?id=185609021

Any lines you may need to configure in the sv_init file are commented with explanations.
The rest should work out of the box so if you don't know what you're doing, you should probably avoid changing anything else.
